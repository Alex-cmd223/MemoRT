Projecto para el proyecto de 4to semestre.
Videjuego "MemoRetos" 
Videjuego basado en grafos hecho con figuras geometricas interpuestas en donde la suma de cada nodo de cualquier figura de una suma.
Dicha suma tiene que ser igual a la suma de los nodos de cualquier otra figura.
Se utiliza Unity, MySql, React y Python.
